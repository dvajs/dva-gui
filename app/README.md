# dva-gui

A gui tool for dva.

> Under development.

## Getting Started

First, clone dva-ast.

```bash
$ git clone github/repo/of/dva-ast
$ cd dva-ast
$ npm install
$ npm run dev
```

Then, go to dva-gui, install dependencies, `npm link` dva-ast and run build in watch mode.

```bash
$ cd path/to/dva-gui
$ npm install
$ npm link path/to/dva-ast
```

Finally, open another terminal tab and run:

```bash
$ cd path/to/dva-gui
$ npm run dev
```
